## Overview

**BhavVani** is a Hindi Speech Emotion Recognition (SER) dataset developed to facilitate research in affective computing and speech-based emotion classification.

This repository currently provides a **sample subset** of the dataset for demonstration and research purposes.

---

## Dataset Details

| Attribute           | Description                   |
|---------------------|-------------------------------|
| **Language**        | Hindi                         |
| **Task**            | Speech Emotion Recognition    |
| **Data Type**       | Audio                         |
| **Format**          | `.wav` (recommended)          |
| **Classes**         | Angry, Happy, Neutral, Sad    |
| **Annotation Type** | Supervised (emotion-labelled) |

---

## Dataset Structure

```
SRCASW-BhavVani/
 ├── Angry/
 ├── Happy/
 ├── Neutral/
 └── Sad/
```

Each directory contains speech samples corresponding to a specific emotional class.

## Applications

- Speech Emotion Recognition (SER)
