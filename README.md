# 🎧 BhavVani: Hindi Speech Emotion Recognition Dataset

## 📌 Overview
BhavVani is a Hindi Speech Emotion Recognition (SER) dataset developed for research in speech-based emotion classification.

This repository contains a **sample version** of the dataset.

---

## 📊 Dataset Details
- Language: Hindi  
- Data Type: Audio (.wav)  
- Task: Speech Emotion Recognition  
- Classes:
  - Angry
  - Happy
  - Neutral
  - Sad  

---

## 📂 Dataset Structure
```
SRCASW-BhavVani/
 ├── Angry/
 ├── Happy/
 ├── Neutral/
 └── Sad/
```
